# Personalakten fuer Zeugniserstellung

Diese zehn Personalakten enthalten die Rohdaten und HR-Notizen, die ein Arbeitgeber typischerweise vor der Zeugniserstellung zusammentraegt. Sie sind die Eingabe fuer das Plugin arbeitszeugnisgenerator. Jede Personalakte ergibt mit dem Werkstatt-Prompt ein vollstaendiges qualifiziertes Zeugnis.
