# Testakten — Bauträgervertragsprüfer

Dieser Ordner enthält zwei Musterverträge zum Ausprobieren des Skills.

| Akte | Charakter |
| --- | --- |
| [`bautraegervertrag/`](./bautraegervertrag/) | Bewusst fehlerhaft — viele rote Ampeln, unwirksame und nichtige Klauseln |
| [`bautraegervertrag-marewald/`](./bautraegervertrag-marewald/) | Rechtmäßig, aber grenzwertig verkäuferfreundlich — überwiegend orange, keine Pflichtsverletzung |

Beide Verträge sind keine Musterverträge und dürfen nicht als Vorlage in der Praxis verwendet werden. Für jede fachliche Bewertung ist eine eigenständige rechtliche und technische Prüfung erforderlich.
